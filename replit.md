# Conv AI ‚Äî Document Review and Design Execution

A static React + Vite frontend UI exported from Figma. Styled with Tailwind CSS v4, shadcn/radix-ui components, and MUI.

## Running the app

```
npm install
npm run dev
```

The dev server starts on port 5000. Open the preview to explore the UI.

## Pages

- `/chat` ‚Äî AI chat interface (default)
- `/images` ‚Äî Image generation
- `/library` ‚Äî File library
- `/projects` ‚Äî Projects list
- `/builder` ‚Äî Builder view
- `/tts` ‚Äî Text to speech
- `/voice` ‚Äî Voice interface
- `/notifications`, `/account`, `/settings`, `/help`

## Stack

- React 18 + React Router 7
- Vite 6
- Tailwind CSS v4
- Radix UI / shadcn components
- MUI (Material UI)
- Recharts, Framer Motion, React DnD

## Notes

- This is a static UI ‚Äî no backend or AI APIs are connected.
